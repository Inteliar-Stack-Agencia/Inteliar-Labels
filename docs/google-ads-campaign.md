# Campaña Google Ads — Inteliar Labels

Plan inicial para soft launch. Presupuesto: **US$5/día** (~US$150/mes). Geografía: **Argentina**.

## Estructura de campaña

**Tipo:** Search (Red de Búsqueda), no Display.

**Objetivo de conversión:** registro en `/auth/register` (evento `sign_up` en GA4 — ver `docs/`/lib/analytics.ts). Importar como conversión en Google Ads desde GA4 (Herramientas → Conversiones → Importar → Google Analytics 4).

## Grupo de anuncios: Integraciones (arrancar solo con este)

Palabras clave (frase):
- `etiquetas mercado libre`
- `imprimir etiqueta mercado envios`
- `etiquetas tiendanube`
- `integracion tiendanube etiquetas`
- `software etiquetas termicas`
- `alternativa bartender`

## Configuración de puja

- Estrategia: **Maximizar clics** con límite de CPC US$0.30–0.40 (con este presupuesto, "Maximizar conversiones" no tiene suficiente data las primeras 2-3 semanas).
- Estimado: 12-16 clics/día.

## Configuración de cuenta

- Ubicación: Argentina (todo el país)
- Idioma: Español
- Red: **Solo Búsqueda** (desactivar Red de Display y Socios de búsqueda)
- Programación: todo el día al principio

## Negativas mínimas

`gratis descarga crackeado`, `curso`, `trabajo`, `empleo`, `pdf gratis`, `plantilla word`
